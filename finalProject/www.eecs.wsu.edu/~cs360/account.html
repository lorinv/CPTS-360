YOUR Account on cs360.eecs.wsu.edu Server 
1. YOUR WEBSITE at cs360.eecs.wsu.edu :
   Each of you has a user account on the Linux server machine cs360.eecs.wsu.edu

        login name = YOUR last name
        password   = YOUR WSU ID number

2. login to your account by

         ssh YOURNAME@cs360.eecs.wsu.edu
             enter your password

   As of now, YOUR home directory contains:

        public_html --- index.html
                     |- ASS1 : for submitting assignment #1
                     |- ASS2 : for submitting assignment #2
                       (etc)

NOTE: The server machine is NOT administered by WSU systems people.
      If you have any problem with login or webpage, please contact KCW directly.

3. Access YOUR website by http://cs360.eecs.wsu.edu/~YOURNAME
   You will get a webpage (by the default index.html file), which looks like

        ------------------------------------
        |    Welcome to YOUR Website       |
        |       ------------------         |
        |       | YOUR PICTURE   |         |
        |       ------------------         |
        | Input command  :  BOX            |
        | Input filename1:  BOX            |
        | Input filename2:  BOX            |
        | Submit command :  Submit BOX     |
        ------------------------------------

4. You may NOT like your current picture.
       login to your account.
       Edit the index.html file to change it to YOUR pretty picture.

5. Submit your assignment to your account ON OR BEFORE THE DUE DATE:

      scp FILENAME YOURNAME@cs360.eecs.wsu.edu:public_html/ASS_number
